# moddy

### 💬 **모디**는,

포트폴리오를 위해 헤어모델이 필요한 **헤어디자이너**와 원하는 머리를 부담없이 하고싶은 **일반인 모델**을 연결하는 플랫폼입니다.

**지출 없이 예쁜 머리**, 모디 <br/>
‘**머리’라는 도화지에 마음껏 꿈을 펼칠 수 있도록,** 모디 <br/>
`#쉬운` `#믿을 수 있는`
